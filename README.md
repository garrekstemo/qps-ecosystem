# QPS Ecosystem Showcase Site

Static overview site for the QPS spectroscopy Julia package ecosystem — the collection of open-source Julia libraries (CurveFitModels, JASCOFiles, SpectroscopyTools, CavitySpectroscopy, ElabFTW) paired with the Quantum Photo-Science Lab's private integration layer (QPSTools, QPSDrive, QPSLab). Deployed via GitHub Pages at <https://garrekstemo.github.io/qps-ecosystem/>.
